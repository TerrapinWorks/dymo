How to Use the Label Maker:

  1. Go to the job page in the Makerbot platform (Jobs -> History)
  2. Select the job for which you would like to print a label
  3. Save the job page using Ctrl + S. It does not matter what you name the file
  4. Click on the "Choose File" button below and select the file
  5. Click upload. The file will be uploaded to the printer and your label will be printed
